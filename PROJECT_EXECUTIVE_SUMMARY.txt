================================================================================
1. PROJECT OVERVIEW
================================================================================

SPECTER addresses the critical gap between Indian citizens and the complex 
legal system by providing instant, accessible legal guidance through:
- AI-powered chatbot (200+ topics, 15+ legal domains)
- Document analysis (OCR + NLP for PDF/Image processing)
- Legal services directory (100+ government portals)
- Lawyer connectivity platform

KEY INNOVATION: The AI does NOT invent answers — it retrieves validated 
information from curated legal datasets and processes it with NLP, fuzzy 
logic, and structured response formatting.

2. PROBLEM & SOLUTION
================================================================================

PROBLEMS:

================================================================================
3. TECHNICAL ARCHITECTURE
================================================================================

STACK:
Frontend: React 18 + TypeScript + Vite
Backend: FastAPI + Python 3.10 + Gunicorn
Database: MongoDB Atlas (user data) + JSON (legal knowledge base)
Deployment: Netlify (frontend) + Render (backend) + Docker
Cost: $0/month (free tier optimization)

CORE ALGORITHM: Hybrid Fuzzy Search
- Exact match (O(1) lookup) → Levenshtein distance → Token overlap
- Response time: < 10ms
- Memory: < 50 MB (vs 2GB+ for vector RAG)
- Accuracy: 88-94%

AI MODELS:
1. Fuzzy Matching (custom-built for legal queries)
2. Tesseract OCR (LSTM-based, 95% accuracy for printed text)
3. NLP (NLTK + custom logic for summarization)

8. CHALLENGES & SOLUTIONS
================================================================================

Challenge 1: SMTP Blocked on Render Free Tier
Solution: Implemented HTTP webhook + SendGrid API fallback + console logging

Challenge 2: Out-of-Memory with Vector RAG
Solution: Replaced with custom fuzzy matching (50MB vs 2GB)

Challenge 3: Mobile UI Clutter
Solution: Mobile-first redesign (hidden subscription button, centered title)

Challenge 4: bcrypt Version Conflict
Solution: Pinned to version 4.2.1 in requirements.txt
